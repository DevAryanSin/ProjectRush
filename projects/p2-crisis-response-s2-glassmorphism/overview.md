CrisisSync is a Next.js 16 web application designed for rapid, coordinated emergency response in hospitality venues. It eliminates fragmented communication during high-stakes incidents by generating real-time, role-specific crisis coordination checklists. Users input incident details, and Gemini AI instantly provides prioritized actions, assigned responsibilities, and escalation triggers, bridging the gap between distressed individuals, staff, and emergency services. The UI features a polished Glassmorphism design with an urgent red and cool white palette for intuitive and efficient crisis management.
