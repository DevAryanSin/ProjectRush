# Brief
Modern global supply chains are inherently volatile, facing chronic disruptions from weather events to operational bottlenecks often identified too late. DisruptRadar addresses this by providing a proactive, AI-driven solution for real-time supply chain risk assessment and dynamic mitigation, ensuring timely and optimized logistics.

# Opportunities
## Differentiation
DisruptRadar differentiates itself through its AI-powered preemptive detection and specific, actionable mitigation recommendations, moving beyond reactive disruption management. The unique Bento Box Grid UI offers a visually engaging and information-rich user experience, unlike standard dashboard layouts.

## Problem Solving Approach
The system continuously analyzes multifaceted transit data, including weather patterns, geopolitical factors, infrastructure reliability, seasonal demand, and historical disruption data. It formulates dynamic mechanisms to instantly execute or recommend highly optimized route adjustments, preventing localized issues from cascading.

## USP
"Input your shipment route — get a disruption risk score before it happens." DisruptRadar's core USP is its comprehensive, AI-driven, real-time disruption prediction coupled with actionable, data-driven mitigation strategies, all presented within an intuitive and information-dense interface.

# Features
- User input form for origin, destination, cargo type, departure date, and transport mode.
- AI-generated disruption probability score (0-100%) for specific routes.
- Identification and scoring of top 3 risk factors.
- Severity breakdown by categories (weather, geopolitical, infrastructure, demand).
- 2-3 specific and actionable mitigation recommendations.
- Dynamic loading states during AI analysis.
- Graceful error handling for API calls.
- Fully responsive Bento Box Grid UI for desktop and mobile.
