VolunteerMatch is a Next.js application designed to revolutionize volunteer coordination for NGOs. It addresses the challenge of fragmented community needs data by providing a centralized system to gather information and intelligently match volunteers to urgent tasks using Gemini AI. Users input volunteer profiles and task descriptions, receiving a comprehensive match score, detailed reasoning, and actionable recommendations. The platform features a professional, glassmorphism UI with a warm green and earthy orange palette, ensuring an intuitive and engaging user experience.
