# Brief
Modern supply chains face chronic disruptions, often identified too late. BottleneckBot addresses this by providing a scalable, AI-powered system for preemptive detection and diagnosis of transit disruptions. It aims to prevent minor issues from escalating into major delays, ensuring operational continuity and efficiency.

# Opportunities
## Differentiation
BottleneckBot distinguishes itself through its real-time, AI-powered diagnostic capabilities, offering a proactive approach to supply chain management. Its flat design UI prioritizes clarity and efficiency, embodying a logistics-precision aesthetic with bold industrial teal and amber, contrasting with cluttered, legacy systems.

## Problem Solving Approach
The application utilizes a sophisticated AI model to analyze multifaceted transit data and user input. It formulates precise diagnoses of bottlenecks, identifies root causes, quantifies cascade risks, and generates highly optimized route adjustments or interventions before disruptions compromise delivery timelines.

## USP
Pinpoint accuracy in identifying and localizing supply chain bottlenecks, coupled with proactive, AI-driven prescriptive interventions. The intuitive flat design interface ensures immediate understanding and actionability of complex diagnostic information.

# Features
- AI-powered root cause analysis for supply chain delays.
- Precise bottleneck location identification.
- Cascade risk assessment for downstream operations (HIGH/MEDIUM/LOW).
- Targeted, actionable fix recommendations ordered by impact.
- Responsive, clean user interface with a distinctive flat design aesthetic.
- Secure server-side Gemini API integration for robust performance.

# Technologies
- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS + Custom CSS Variables
- **Icons**: lucide-react
- **AI**: Google Gemini 2.5 Flash API (via `gemini-2.5-flash`)
