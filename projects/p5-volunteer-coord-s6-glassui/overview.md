This solution provides a Next.js 16 web application for UrgencyRank, a tool designed to analyze and prioritize community needs based on user-provided data. It uses Gemini AI to process freeform community needs, available volunteer counts, budget, and timeframes, outputting a ranked list of needs with urgency scores, resource allocation recommendations, and a deployment strategy. The application features a distinctive Glass UI style with dark themes and neon accents, prioritizing a professional and polished user experience.
