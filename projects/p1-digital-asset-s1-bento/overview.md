This solution provides a functional MVP for ScanGuard, a digital asset protection tool for sports media. It's a Next.js 16 application featuring a Bento Box Grid UI. Users input a URL or content description, which is sent to a server-side Gemini AI endpoint. The AI analyzes the content for unauthorized use, returning a structured verdict. The application then parses and displays the risk level, confidence score, red flags, and recommended actions in an intuitive, responsive dashboard.
