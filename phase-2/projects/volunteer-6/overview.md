This solution is a Next.js 16 web application, EventPlanner, designed to streamline volunteer event planning for NGOs and social groups. It takes user-defined community needs and available resources as input and leverages the Gemini API to generate a comprehensive, actionable event plan. The application features a playful, claymorphism-inspired UI with a pastel color palette, rounded elements, and generous whitespace, ensuring a user-friendly and engaging experience.
