This project delivers a Next.js 16 web application named PostMortemPro, designed for the hospitality industry. It simplifies crisis management by allowing users to input details about an incident. The application then leverages the Gemini API to automatically generate a comprehensive post-mortem report, structured for clarity and actionability. The UI adheres to a strict 'Paper/Editorial' style, emphasizing readability and a professional, trustworthy aesthetic with serif fonts and a limited color palette.
