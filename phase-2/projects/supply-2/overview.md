This solution implements a Next.js 16 web application named "VendorScore" that leverages the Gemini API to generate vendor reliability scorecards. It features a Neumorphism UI style with a soft color palette of white and forest green, offering a user-friendly interface for inputting vendor details and displaying comprehensive reliability assessments, including scores, grades, risk flags, and recommendations. The application is designed to be fully responsive and handles loading and error states gracefully.
