This solution provides a Next.js 16 MVP web application called SkillGapFinder. It addresses the problem of scattered community data by allowing NGOs to input project details and available volunteer skills. The application leverages the Gemini API to analyze skill gaps, identify critical missing skills, quantify their impact, and generate targeted recruitment briefs and interim workarounds. The UI is styled with a distinctive "Paper/Editorial" aesthetic for trustworthiness and readability.
