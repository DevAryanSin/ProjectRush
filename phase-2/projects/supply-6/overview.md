This solution provides a Next.js 16 MVP for DemandSignal, a tool that forecasts near-term demand shifts based on market signals. It features a Claymorphism UI, client-side form submission to a server-side API route that integrates with the Gemini API for AI-driven predictions. The application is fully responsive, handles loading and error states gracefully, and presents Gemini's analysis in a visually structured and user-friendly format.
