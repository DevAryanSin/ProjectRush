This project delivers a Next.js 16 web application called PromptBias, designed to detect and mitigate bias within AI prompts. It features a striking Brutalist UI, providing users with a clear interface to input prompts, receive detailed bias analyses (including bias types, affected groups, severity, and rewritten prompts), and leverage Gemini's powerful AI for accurate assessments. The application prioritizes accessibility and provides a critical tool for algorithmic accountability.
