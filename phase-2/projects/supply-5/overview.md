This solution provides a Next.js 16 web application named "ContractClause" designed to generate protective contract clauses for supply chain disruption scenarios. It employs a sophisticated Paper/Editorial UI style, using serif fonts and a structured layout to convey trustworthiness. The application integrates with the Gemini API to draft legally-sound clauses covering force majeure, SLA modifications, penalty limits, and dispute resolution, addressing the critical need for proactive risk management in global supply chains.
